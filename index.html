sm.regui

Imagine this, you've literally remade MyGui inside MyGui, and somehow the inner one works better than the outer one. That's what sm.regui is


sm.regui is a awesome GUI library for Scrap Mechanic which uses a exploit to write XML code via lua to be able to create MyGui Layout Files. It has features that MyGui itself doesn't have. Such as:

  • Creating, destroying and modifing widgets

  • Custom Font Rendererer

  • Fullscreen GUI system


There are more features however you should check it's github out for that.


Steam page Website

VSMX

VSMX is a massive project of mine which basicly is a SDK for Scrap Mechanic allowing modders to do whatever they please. VSMX will let you:

  • Control the renderer

  • Perform complicated tasks with Multi-threading

  • Full-blown GUI library (ImGui & sm.regui reimplementations for GUIs)


There will be more features that VSMX would have however its private due to compeititors. VSMX will be finished around 2026 or 2027 (this is not a promise)
